algo-go
=============

`algo-go` - implemetation of different alghoritms with `golang`.

Alghoritms
----------

#### Sorting

  * Bubble sort
  * Selection sort
  * Merge sort
  * Quick sort
  * Shell sort
  * Cocktail sort

###### not implemented

  * Heap sort
  * Counting sort

#### Searching

  * Minimax
  * Negamax

###### not implemented

  * AlphaBeta
